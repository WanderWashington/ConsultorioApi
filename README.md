# Spring Boot 3 CRUD

API REST desenvolvida com Spring Boot 3 durante estudos da Alura.

## Tecnologias
- Java 21
- Spring Boot 3
- Maven
- Flyway
- MySQL
- JPA/Hibernate

## Executar projeto

```bash
.\mvnw.cmd spring-boot:run
```

## Rodar migrations

As migrations são gerenciadas pelo Flyway automaticamente ao iniciar a aplicação.
